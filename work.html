Design principles:

Responsive

Design that allows great experiences on multiple devices is expected nowadays. I always give love both to the smaller devices and the desktop.

User focus

Understanding the user is the key of making a usable website. Therefor good research and testing is an good investment. This is something I build my designs around, striving for a better end user experience.

Performance

Development with performance as focus lead to faster loading time. To optimize images, minify code and remove useless code all contributes to a better website.
