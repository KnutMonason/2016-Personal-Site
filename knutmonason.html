Website


Description

This is my personal website. It’s mainly intended for people that are already familiar with web development to showcase some of my work and skillsets in collaboration purposes.

Process

For this project I used Bootstrap grids, Jquery interactions and some custom Javascript for the moving background on the landing page. I mainly became static because of the server capabilities on this domain, but it’s fun working under some restrains.

Principles

Something that I had in mind when I designed the structure of the website was to develop a flow through the content. Aiming for a very linear experience while allowing the user to jump to the most interesting part right away. It also have highlighted words here and there to quickly direct the user to what I think is the more important things.
